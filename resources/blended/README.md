# Generate trigger

<b>CIFAR-10:</b><br/>
32x32 image with 3 color channels<br/><br/>
<b>MNIST:</b><br/>
28x28 image with 1 color channel<br/><br/>

```shell
python ./generate_random_pattern.py --output_path ./random_pattern.png
```
